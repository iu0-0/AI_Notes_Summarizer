# 🎙️ AI Notes Summarizer
 
A Streamlit app that converts audio recordings into structured notes using Groq's Whisper API for transcription and LLaMA 3 for summarization.
 
---
 
## Features
 
- Upload audio files (WAV, MP3, M4A, WEBM)
- Automatic speech-to-text via Groq Whisper
- Structured notes generated by LLaMA 3.1
- Outputs: Main Topics, Key Points, Action Items, Summary
- Download transcript and notes as `.txt` files
- Soft pixel UI theme, dark and light mode friendly

- Python 3.10 or higher

## Local Setup
 
**1. Clone the repo**
 
```bash
git clone https://github.com/yourusername/AI_Notes_Summarizer.git
cd AI_Notes_Summarizer
```
 
**2. Create and activate a virtual environment**
 
```bash
python -m venv venv
 
# Windows
venv\Scripts\activate
 
# macOS / Linux
source venv/bin/activate
```
 
**3. Install dependencies**
 
```bash
pip install -r requirements.txt
```
 
**4. Add your API key**
 
Create a `.env` file in the root folder:
 
```
GROQ_API_KEY=your_groq_api_key_here
```
 
**5. Run the app**
 
```bash
streamlit run app.py
```
 
The app will open at `http://localhost:8501`

## Deployment
 
### Hugging Face Spaces (recommended)
 
1. Create a new Space at https://huggingface.co/new-space and select **Streamlit** as the SDK
2. Push your code:
```bash
git clone https://huggingface.co/spaces/yourusername/AI-Notes-Summarizer
cp app.py requirements.txt AI-Notes-Summarizer/
cd AI-Notes-Summarizer
git add .
git commit -m "deploy"
git push
```
 
3. Go to **Settings → Repository Secrets** and add:
   - Name: `GROQ_API_KEY`
   - Value: your key
